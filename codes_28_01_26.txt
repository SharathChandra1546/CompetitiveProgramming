You are given an integer n.
Return all numbers in the range 1 to n whose binary representation contains 
only set bits (1s).

Example:
Input = 7
Output = 1 3 7
import java.util.*;
class CompleteSetBits{
    public static int len(int n){
        int r = n;
        int l = 0;
        while(r>0){
            r=r>>1;
            l++;
        }
        return l;
    }
    public static void allSet(int n){
        int count =0;
        int r = n;
        while(r>0){
            r &=r-1;
            count++;
        }
        if(len(n)==count){
            System.out.print(n+",");
        }
    }
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for(int i=1;i<n+1;i++){
            allSet(i);
        }
    }
}


You are given an integer n.
Your task is to print all numbers from 0 to n (inclusive) whose binary 
representation is a palindrome.

Example:
n=7
Output is 0 1 3 5 7

import java.util.*;
class BinaryPalindrome{
    public static boolean palindrome(int n){
        String s = Integer.toBinaryString(n);
        int left = 0, right = s.length()-1;
        int res = 0;
        while(left<=right){
            if(((s.charAt(left))^(s.charAt(right)))!=0) return false;
            left++;
            right--;
        }
        return true;
    }
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for(int i=0;i<=n;i++){
            if(palindrome(i)){
                System.out.print(i+",");
            }
        }
    }
}

You are given a string S.
Your task is to check, can any one of the AlphaCipher is a palindrome or not.

Input Format:

A string S

Output Format:
--------------
Print a boolean value.


Sample Input-1:
---------------
carrace

Sample Output-1:
----------------
true


Sample Input-2:
---------------
code

Sample Output-2:
----------------
false

import java.util.*;
public class PalindromePermutation{
    public static boolean palindrome(String s){
        int bitmask = 0;
        for(char ch:s.toCharArray()){
            int bit = 1<<(ch-'a');
            bitmask = bitmask^bit;
        }
        
        return (bitmask==0) || (((bitmask-1)&bitmask)==0);
        
    }
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine();
        System.out.print(palindrome(s));
    }
}
